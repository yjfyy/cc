1. Installation/Deinstallation

Installieren: 

Legen Sie die Feuersturm-CD ein. 
Wählen Sie Installieren. 
Wenn der Autostart-Bildschirm nicht automatisch erscheint:
Doppelklicken Sie auf das "Arbeitsplatz"-Icon. 
Doppelklicken Sie auf das CD-Icon. 
Doppelklicken Sie auf das Setup-Icon.
Folgen Sie während der Installation den Anweisungen auf dem Bildschirm. 

Anders als für Operation Tiberian Sun wird für Feuersturm keine Seriennummer zur Installation benötigt.


1.1 DirectX-Informationen

Command & Conquer Teil 3: Operation Tiberian Sun ist eine Microsoft DirectX-Anwendung. 
Version 6 von Microsoft DirectX ist auf der CD enthalten, und Sie haben die Option, diese während der Installation von Command & Conquer Teil 3: Operation Tiberian Sun mit zu installieren.

Bei einigen frühen Versionen von Windows 95 kann DirectX 6 nicht direkt von der CD Command & Conquer Teil 3: Operation Tiberian Sun installiert werden.
Wenn das passiert, starten Sie bitte die Datei DXSetup.exe, die sich im Ordner DXSetup auf der CD befindet.
 
Alle Microsoft DirectX-Treiber befinden sich im Ordner DXSetup im Quellverzeichnis der CD. Um die Microsoft DirectX-Treiber neu zu installieren, gehen Sie in den DXSetup-Ordner und starten Sie DXSetup.exe. Sie können die CD durchsuchen, indem Sie im Autostart-Titelbildschirm die Option "CD durchsuchen" wählen. Wenn Sie Probleme mit Windows 95/98 haben, nachdem diese neuen DirectX-Treiber von Microsoft installiert wurden, wenden Sie sich bitte an Ihre örtliche Microsoft-Vertretung. 

1.2 Windows NT-Benutzer 

Damit Command & Conquer Teil 3: Operation Tiberian Sun mit Windows NT 4 funktioniert, muss mindestens Service Pack 3 installiert sein. Bitte wenden Sie sich an den Hersteller Ihres Computers oder schauen Sie unter www.microsoft.com nach, wenn Sie Service Pack 3 benötigen.


1.3 Deinstallation

Sie können das Spiel von der Systemsteuerung aus deinstallieren:
Klicken Sie auf "Start".
Markieren Sie "Einstellungen". 
Klicken Sie auf "Systemsteuerung". 
Doppelklicken Sie auf "Software". 
Wählen Sie "Tiberian Sun" und klicken Sie auf entfernen.
Oder mit dem Autostart-Feature:
Legen Sie die Feuersturm-CD in Ihr CD-ROM-Laufwerk ein. 
Wenn das Autostart-Fenster erscheint, klicken Sie auf deinstallieren. 
Folgen Sie den Anweisungen auf dem Bildschirm. 



2. Problemlösung


2.1 Probleme mit Grafik- und Soundkarten 

In Feuersturm wird die Microsoft DirectX-Technologie für Sound und Grafik eingesetzt. 
Das heißt, dass Ihre Grafik- und Soundkartentreiber 100%ig zu DirectX kompatibel sein müssen. Bei den meisten neuen Computern ist DirectX bereits vorinstalliert. Versuchen Sie, Feuersturm zu starten, bevor Sie DirectX installieren, um zu sehen, ob Ihr Computer bereits DirectX-kompatible Treiber hat. Wenn nicht, können Sie DirectX 7 von der Feuersturm-CD installieren oder sich die aktuellsten DirectX-Treiber unter www.microsoft.com besorgen. Es ist wichtig, dass Sie DirectX neu installieren, nachdem Sie Ihre Grafik- und Soundkartentreiber aktualisiert haben. DirectX 6 ist auf der CD Operation Tiberian Sun enthalten.

Die meisten Sound- und Grafikkarten funktionieren mit den DirectX-Treibern auf der CD, wenn Sie jedoch trotzdem Probleme haben, wenden Sie sich an den Hersteller Ihrer Sound- oder Grafikkarte, um die aktuellsten Treiber für Ihre Hardware zu erhalten. 

Anzeichen dafür, dass Sie Ihre Treiber aktualisieren müssen:
Sie erhalten DirectDraw- oder DirectSound-Fehler. 
Das Spiel stürzt ab oder bleibt hängen. 
Der Sound fällt während des Spiels aus. 
Der Windows-Desktop ist verzerrt oder fehlerhaft. 
 
Wenn Sie keinen Internet-Zugang haben, wenden Sie sich an den Hersteller Ihrer Sound- oder Grafikkarte, um die aktuellsten DirectX-Treiber für Ihre Hardware zu erhalten. Folgen Sie dann den Anweisungen des Herstellers zur Installation der Treiber. 

Soundprobleme (nach dem Aktualisieren der Treiber): 
Vergewissern Sie sich, dass Sie die aktuellsten Updates für Windows haben, einschließlich des aktuellsten Service Packs von Microsoft. Überprüfen Sie Ihr CD-ROM-Laufwerk. 
Der Sound wird von der CD geladen. Wenn das CD-ROM-Laufwerk nicht richtig funktioniert, kann das Soundprobleme verursachen.

Die Monitor-Einstellungen müssen möglicherweise nach der Installation der Microsoft DirectX-Grafiktreiber angepasst werden. Wenn der Bildschirm verzerrt oder horizontal oder vertikal verschoben ist, können Sie ihn mit den Knöpfen am Monitor wieder richtig einstellen.  

Wenn Sie die Option "An Auflösung anpassen" nicht verwenden können, heißt das möglicherweise, dass Ihre Grafikkarte dieses Feature nicht unterstützt.

Wenn Sie das Microsoft Theme Pack verwenden und einen eigenen Cursor gewählt haben, können Grafikprobleme auftreten. Die besten Ergebnisse erhalten Sie, wenn Sie beim Spielen von Command & Conquer Teil 3: Operation Tiberian Sun den Standard-Cursor verwenden.

Bei alten SoundBlaster AWE 64- und AWE 32-Soundkartentreibern können Soundprobleme auftreten, wodurch Command & Conquer Teil 3: Operation Tiberian Sun viel langsamer als üblich läuft. Wenn Sie eine dieser Soundkarten haben und diese Symptome bei Ihnen auftreten, besorgen Sie sich bitte die aktuellsten Treiber vom Hersteller Ihrer Karte.

Unregelmäßiger Sound bei bestimmten Systemen unter Windows NT. Um dieses Problem zu beheben, müssen Sie die aktuellsten Soundtreiber für Ihre Soundkarte installieren.

Die meisten Soundkartenprobleme können behoben werden, indem die aktuellsten Treiber für die jeweilige Karte installiert werden. Hinweise dazu, wie Sie die aktuellsten Treiber erhalten, finden Sie in der Dokumentation zu Ihrer Soundkarte.

Wenn Sie USB-Lautsprecher benutzen, schalten Sie die Lautsprecher während des Spiels nicht aus, denn das kann zu Abstürzen führen.

Wenn eine andere Anwendung die Soundkarte während der Installation oder eines Spiels benutzt, erscheint eine Fehlermeldung "DXSound Error". Wenn dieser Fehler auftritt, schließen Sie bitte alle Anwendungen und starten oder installieren Sie das Spiel erneut.


2.1.1 Soundprobleme unter Windows 2000 

Benutzer von Windows 2000 haben möglicherweise Probleme mit dem Sound bei Feuersturm. Zunächst können Sie versuchen, einige der Leistungsparameter in der Multimedia-Systemsteuerung zu verändern. 
Wählen Sie im Start-Menü Einstellungen und dann Systemsteuerung. Doppelklicken Sie auf das Multimedia-Icon. In der Registerkarte Audio klicken Sie auf Weitere Eigenschaften unter Wiedergabe. In der Registerkarte Leistungsmerkmale können Sie die Beschleunigungseinstellungen nach der Empfehlung von Windows verändern.

Am 18. Januar 2000 unterstützte Windows 2000 die meisten Creative Labs-Soundkarten (einschließlich der SoundBlaster-Reihe) noch nicht korrekt.
Creative Labs gibt aktualisierte Treiber für seine Produkte heraus, mit denen die meisten oder alle Inkompatibilitätsprobleme behoben werden sollten, aber bis diese Treiber verfügbar sind, können die Benutzer ihre SoundBlaster-Karten möglicherweise nicht unter Windows 2000 benutzen.

2.5 Probleme mit Virtuellem Speicher/RAM 

Für Command & Conquer Teil 3: Operation Tiberian Sun wird möglicherweise mehr RAM benötigt als auf Ihrem Computer vorhanden ist. Windows 95/98 kümmert sich mit dem sogenannten "Virtuellen Speicher" automatisch um dieses Problem. Hiermit wird Speicherplatz auf Ihrer Festplatte benutzt, um den benötigten Speicher zu simulieren und die Daten auf der Festplatte hin- und herzuschieben. Unter Windows 95/98 können Sie die Größe des Festplattenplatzes für den Virtuellen Speicher selbst bestimmen. WIR EMPFEHLEN IHNEN JEDOCH, DAS NICHT ZU TUN! Um den Virtuellen Speicher von Windows 95/98 verwalten zu lassen, gehen Sie in das Windows 95/98-Start-Menü und wählen Sie 'Einstellungen\Systemsteuerung'. Dann doppelklicken Sie auf 'System' und klicken auf 'Leistungsmerkmale\Dateisystem\Virtueller Arbeitsspeicher'. Wählen Sie dann 'Windows verwaltet die Einstellungen für den Virtuellen Speicher (empfohlen)'. 

In bestimmten Computern befinden sich Grafikkarten, die Systemspeicher als Grafikspeicher benutzen. Auf diesen Computern läuft Command & Conquer Teil 3: Operation Tiberian Sun möglicherweise nicht, wenn Sie nur 32 MB Speicher haben.
Da die Grafikkarte Systemspeicher benutzt, normalerweise 4 oder 8 MB, bleibt auf Computern mit 32 MB nicht genug Speicher übrig, damit das Spiel gestartet werden kann. Damit Command & Conquer Teil 3: Operation Tiberian Sun korrekt läuft, müssen Sie Ihren Systemspeicher vergrößern oder eine Grafikkarte mit eigenem Grafikspeicher benutzen.


2.6 Andere Probleme 

2.6.1 Systemabstürze und Neustarts 
Vergewissern Sie sich, dass Sie die aktuellsten Windows-Updates haben.
Überprüfen Sie Ihre CD-ROM-Einstellungen. 
Vergewissern Sie sich, dass Sie die Grafik- und Soundkartentreiber aktualisiert haben. 

2.6.2 Autostart
Wenn Ihre CD nicht automatisch startet, wenn sie ins CD-ROM-Laufwerk eingelegt wurde, ist dieses Feature möglicherweise deaktiviert. Um Autostart zu aktivieren, rechtsklicken Sie auf 'Arbeitsplatz' und wählen Sie 'Eigenschaften' und dann 'Gerätemanager'. Wählen Sie Ihr CD-ROM-Laufwerk und klicken Sie auf 'Eigenschaften'. Wählen Sie 'Einstellungen' und aktivieren Sie das Kästchen 'Automatische Benachrichtigung beim Wechsel'.

2.6.3 16-Bit CD-ROM-Treiber 
Command & Conquer Teil 3: Operation Tiberian Sun startet möglicherweise nicht, wenn Sie einen 16-Bit CD-ROM-Treiber in der AUTOEXEC.BAT oder CONFIG.SYS haben. Wenn Sie Verweise auf MSCDEX oder ähnliche Treiber haben, dann versuchen Sie, sie in den autoexec.bat- und config.sys-Dateien zu entfernen. Diese Treiber werden normalerweise nicht unter Windows 95/98 benötigt, in den meisten Fällen ist die Leistung ohne sie sogar noch besser.

2.6.4 Fehler bei Filmsequenzen 
Wenn Sie Fehler beim Abspielen von Filmsequenzen erhalten, versuchen Sie, den Zusatz-Cache-Speicher unter Windows 95/98 zu reduzieren, indem Sie Start/Einstellungen/Systemsteuerung öffnen und dann auf das System-Icon doppelklicken. Klicken Sie im folgenden Fenster auf die Registerkarte Leistungsmerkmale und dann auf Dateisystem. Im folgenden Fenster klicken Sie auf die Registerkarte CD-ROM und notieren sich die Größe des Zusatz-Cache-Speichers (damit Sie sie später wiederherstellen können). Stellen Sie jetzt die Größe des Zusatz-Cache-Speichers auf Klein. Das sollte die Probleme mit den Filmsequenzen beheben.

2.6.5 Powersave-Modi 
Es ist möglich, dass Ihr Computer bei längerer Inaktivität die Verbindung zu Ihrem Internet Service Provider verliert. Manchmal wird das durch den Schlafmodus in den "Powersave"-Optionen verursacht. Dadurch können Probleme im Spiel oder bei der Verbindung zu Westwood Online entstehen. Um Probleme zu vermeiden, deaktivieren Sie Ihren Bildschirmschoner oder Ihre Powersave-Modi während des Spiels.

2.6.6 Verlangsamungen 
Wenn es während es Spielens von Command & Conquer Teil 3: Operation Tiberian Sun zu Verlangsamungen kommt, wenn gerade auf die CD zugegriffen wird, liegt das wahrscheinlich an der Musik, die von der CD gespielt wird. Wenn dieses Problem auftritt, installieren Sie Command & Conquer Teil 3: Operation Tiberian Sun bitte neu und installieren Sie die Musikdateien auf der Festplatte. Somit werden die Verlangsamungen beim Zugriff auf die CD stark reduziert.

Wenn es zu Verlangsamungen beim Spielen von Command & Conquer Teil 3: Operation Tiberian Sun kommt und nicht auf die CD zugegriffen wird, vergewissern Sie sich, dass während Ihres Spiels keine anderen Anwendungen laufen. Dies betrifft auch Anti-Virus-Software.



3. Informationen

3.1 Battle-Clans, Turniere und der Weltherrschaftsmodus


3.1.1 Battle-Clans

Erstmals in Tiberian Sun ermöglichen die Battle-Clans Menschen aus aller Welt, sich zusammenzufinden und gegen andere Spielergruppen anzutreten. Jeder Spieler von Tiberian Sun kann einen Battle-Clan erstellen. Das geht ganz einfach!

Entscheiden Sie sich, worauf sich Ihr Battle-Clan spezialisieren soll. Möchten Sie ein Motto haben? Eine Flagge? 
Eine Kampf-Philosopie? Und was am wichtigsten ist: Möchten Sie für die GDI oder für Nod kämpfen?

Wenn Sie diese Informationen für Ihren Clan bestimmt haben, gehen Sie zu http://battleclans.westwood.com/tibsun/ und klicken Sie auf den Button "Create Clan".
Jetzt kommen Sie zum Bildschirm zum Erstellen eines Battle-Clans. 

Füllen Sie die Informationen auf diesem Bildschirm aus, um Ihren Battle-Clan einzurichten. Ein paar wichtige Punkte: Das WOL-Login und das Passwort sind die Account-Informationen für Sie, den Gründer des Battle-Clans.  

Der Gruppenname ist der volle Name des Clans, die Abkürzung taucht bei den Namen der Mitglieder in den Chatrooms von Westwood Online auf.

Das Administrator-Passwort sollte nur an diejenigen weitergegeben werden, von denen Sie möchten, dass sie vollen Administrator-Zugang zum Clan haben. Das schließt auch die Möglichkeit ein, Mitglieder zu entfernen und den Clan-Status zu verändern.

Das Teilnehmer-Passwort sollte an jeden gegeben werden, den Sie im Clan dabeihaben möchten. Dieses Passwort muss von dem neuen Mitglied eingegeben werden, wenn er oder sie versucht teilzunehmen.

Status - open oder closed - steht einfach nur dafür, ob Sie neue Mitglieder annehmen oder nicht.

Wenn Sie eine Clan-Homepage, E-Mail-Adresse oder einen ICQ-Account haben, können diese in den entsprechenden Feldern angegeben werden.

Außerdem kann Ihr Clan ein Motto haben. Dieses Motto kann von allen angeschaut werden, die sich auf der Seite mit der Clan-Liste befinden.

Wenn Sie Ihren Clan erfolgreich erstellt haben, kommen Sie zum Battle 
Clan Flag Creator. Folgen sie hier einfach den Anweisungen auf dem Bildschirm, um Ihre persönliche Clan-Flagge zu erstellen.


Wenn Ihr Clan erstellt ist, können Sie beginnen, Mitglieder dafür zu suchen! Wenn Sie ein neues Mitglied gefunden haben, geben Sie ihm das Teilnehmer-Passwort und die Adresse http://battleclans.westwood.com/tibsun/, damit er/sie die Clan-Liste durchsuchen kann. Von diesem Bildschirm aus kann er/sie an Ihrem Clan teilnehmen, die Westwood Online-Informationen eingeben und auf den Button Join Clan klicken. 

Bitte beachten Sie, dass maximal 20 Spieler in einem Battle-Clan sein können.

Die Verwaltung kann von der Battle-Clan-Seite aus vorgenommen werden, indem man auf den Button "Clan Administration" klickt.

Wenn Sie ein Clan-Spiel starten möchten, wählen Sie "BattleClan", wenn Sie ein Spiel unter Westwood Online erstellen.



3.1.2 Turniere

Jeder Spieler kann an den Feuersturm-Turnieren bei Westwood Online teilnehmen.
Wenn Sie das Spiel erstellen, wählen Sie "Ladder" im Fenster zur Spielerstellung. Wenn Sie an einem Spiel anderer Spieler teilnehmen, vergewissern Sie sich, dass es sich um ein Ladder-Spiel handelt, indem sie sich das Icon anschauen. Wenn es eine Trophäe ist, beeinflussen die Ergebnisse des Spiels Ihre Turnierposition.

Wenn Sie bei Westwood Online sind, finden Sie unten im Bildschirm Links, mit denen Sie direkt zur Website der Ladder Ranking gelangen. Wenn Sie auf diesen Button klicken, wird Feuersturm minimiert, und die Website wird in Ihrem Web-Browser geöffnet. Wenn Sie die Website geladen haben, können Sie sich die aktuelle Rangliste anschauen. Die Ergebnisseite wird alle paar Stunden aktualisiert, so dass die angezeigten Ergebnisse möglicherweise nicht hundertprozentig korrekt sind.

Wenn Sie auf den Namen eines Spielers klicken, können Sie die Statistik dieses Spielers anschauen. Von dieser Seite aus können Sie auch das Ergebnis aller Ladder Games sehen, die der Benutzer gespielt hat. Die Informationen, die angezeigt werden, nachdem Sie auf ein einzelnes Spiel geklickt haben, sind unter anderem die Spieler im Spiel, das Ergebnis sowie einige statistische Informationen über den Kampf.



3.1.3 Der Weltherrschaftsmodus

Der Weltherrschaftsmodus in Feuersturm lässt den Kampf zwischen GDI und Nod auf einer ganz neuen Ebene entstehen. Hier treten GDI- und Nod-Spieler in einem Multiplayer-Spiel um die Kontrolle über Nordamerika oder Europa gegeneinander an. Die Spieler treten unter verschiedenen vordefinierten Bedingungen auf jedem Gelände gegeneinander an. Am Ende bestimmt der Ausgang der Schlacht, welche Partei dieses Territorium kontrolliert. Die Kampflinien zwischen den beiden Parteien werden vor- oder zurückgeschoben, je nachdem wer welches Gebiet erobert hat, immer neue Gebiete werden umkämpft. Nach einiger Zeit gewinnt eine Seite die Oberhand über ein Gebiet und kann sich freuen.

Der Weltherrschaftsmodus ist im Hauptmenü-Menü verfügbar. Wie bei jedem Online-Spiel sollte die Internetverbindung aktiviert sein, bevor Sie auf den Button Weltherrschaft klicken. Wenn Sie sich zum ersten Mal auf dem Weltherrschafts-Server einloggen, können Sie wählen, auf welcher Seite Sie kämpfen möchten - GDI oder Nod. In den folgenden Weltherrschafts-Spielen spielen Sie für die Partei, die Sie anfänglich ausgewählt haben. Diese Auswahl bleibt bis zum Beginn der nächsten Weltherrschafts-Runde bestehen (normalerweise ca. 2 Wochen).

Nachdem Sie Ihre Seite gewählt haben, erscheint die Karte des laufenden Kampfes. Auf der Karte können Sie sehen, wo Nod, GDI und die umkämpften Gebiete liegen, und Sie können wählen, in welchem Gebiet Sie kämpfen möchten.  
In jedem Gebiet gibt es eine große Bandbreite von Spielen.  
Viele Optionen sind vordefiniert, meistens müssen Sie nur wählen, unter welchen Optionen Sie antreten möchten. Wenn Sie den Cursor über ein umkämpftes Gebiet bewegen, sehen Sie eine Liste der Kampfkonditionen für diese Karte.  
Die Technikstufe ist möglicherweise herabgesetzt, oder Sie haben zu Beginn weniger Geld. Manche umkämpfte Gebiete sind schwieriger zu erobern als andere, schauen Sie sich also diese Informationen an, bevor Sie ein Spiel beginnen. Wenn Sie den Cursor über andere GDI- oder Nod-Gebiet bewegen, sehen Sie, wie's in der übrigen Welt aussieht.


3.2 Westwood Online

Westwood Online ist Westwoods Online-Spiel-Server. Hier können Menschen aus aller Welt kostenlos gegeneinander antreten. Tiberian Sun: Feuersturm ist zu Westwood Online kompatibel und ermöglicht Spiele gegen viele menschliche bzw. Computer-Gegner über das Internet.

Wenn Sie im Multiplayer-Menü Internet wählen, kommen Sie zum Login-Bildschirm von Westwood Online (WOL). Wenn Sie bereits einen Account bei WOL haben, geben Sie einfach Ihren Spitznamen und Ihr Passwort ein, dann sind Sie sofort online im Spiel. 
Wenn Sie keinen Account haben, können Sie einen erstellen, indem Sie auf Neuer Account klicken und den Anweisungen auf dem Bildschirm folgen. Um zum Multiplayer-Menü zurückzukehren, klicken Sie auf Abbrechen.

Wenn Sie noch nie über WOL gespielt haben, müssen Sie einen Benutzer-Account erstellen, bevor Sie spielen können. Klicken Sie auf Neuer Account, dann beginnt der Vorgang, mit dem Sie schließlich online spielen können. Wenn Sie sich bei Westwood eingeloggt haben, müssen Sie ein paar Fragen beantworten, bevor ein Account für Sie erstellt wird.

Wenn alles glatt läuft mit der Erstellung des Accounts, versucht das Programm, sich mit WOL und der Spiel-Lobby zu verbinden. Wenn das Programm Schwierigkeiten mit der Herstellung der Verbindung zum Server hat, erscheint eine Nachricht mit dem jeweiligen Problem. Weitere Informationen über potentielle Probleme finden Sie im Abschnitt Problemlösung der Feuersturm-Referenzkarte.

Wenn Sie eine Verbindung zu WOL hergestellt haben, erscheint der WOL-Lobby-Bildschirm. In diesem Bildschirm stehen Ihnen verschiedene Optionen zur Verfügung wie z.B. an einem Spiel teilnehmen, ein neues Spiel einrichten, mit anderen Spielern chatten, andere Spieler suchen und die Turniertabelle anschauen.

Die linke Seite des Chat-Bildschirms ist das Chat-Fenster für eingehende und ausgehende Chat-Nachrichten. Wenn man etwas eintippt, dann erscheinen die Nachrichten so in den Chat-Fenstern, und zwar folgt die Nachricht nach dem Namen dessen, der sie geschrieben hat. Um eine Nachricht zu senden, tippen Sie sie einfach auf die Eingabezeile und drücken die Enter-Taste. Um die Aufmerksamkeit auf eine ausgehende Nachricht zu lenken, können Sie auf den Action-Button auf der rechten Seite klicken. Dann wird die Nachricht in einer anderen Farbe dargestellt, um ihre Wichtigkeit zu unterstreichen. Um eine Nachricht an eine bestimmte Person oder Personengruppe zu schicken, können Sie verschiedene Namen aus der Benutzerliste unten rechts im Bildschirm anklicken.

Weitere Informationen über die Funktion und die Möglichkeiten von Westwood Online finden Sie in der Feuersturm-Referenzkarte.


3.2.1 Update-Informationen
Command & Conquer Teil 3: Operation Tiberian Sun enthält ein automatisches Update-Feature. Sie werden über aktualisierte Versionen informiert, wenn Sie Command & Conquer Teil 3: Operation Tiberian Sun über Westwood Online spielen. Wenn eine aktuellere Version des Spiels verfügbar ist, erhalten Sie sie automatisch. Wenn Sie möchten, wird sie automatisch installiert. Wenn Sie Command & Conquer Teil 3: Operation Tiberian Sun nicht über Westwood Online spielen möchten, dann schauen Sie unter HTTP://www.westwood.com nach Update-Informationen nach.


4. Spielprobleme 

Ingenieure können wie jede andere Einheit in den Wach-Modus versetzt werden. Wenn sie im Wach-Modus sind, reparieren sie automatische jedes in der Nähe befindliche Gebäude des Spielers, das stark beschädigt wurde, oder erobern gegnerische Gebäude in der Nähe.


5. Übersicht der Veränderungen 

Änderungen in V. 2.0
-------------------

-  Tiberian Sun: Feuersturm

Änderungen in V. 1.17a-------------------

- Der Fehler, dass Gebäude an beliebigen Orten gebaut werden konnten, wurde behoben.

Änderungen in V. 1.17
------------------

-  Der Fehler, dass am Ende von Multiplayer-Spielen die Verbindung unterbrochen wurde oder die Daten zu langsam verschickt wurden, wurde behoben.

-  Mit der neuen 'Multi-Ingenieur'-Option für Multiplayer-Spiele verhalten sich die Ingenieure wie in Alarmstufe Rot. Zur Eroberung eines Gebäudes werden dann drei Ingenieure benötigt.

-  Der Fehler, der interne Fehler oder Verbindungsverluste in Multiplayer-Spielen verursachte, wurde behoben.

-  Der Railgun-Schusszwang beschädigt die Gebäude jetzt nicht mehr mehrfach.

-  Auf der Server-Liste erscheinen keine Server mehr doppelt.

Änderungen in V. 1.16a.1
---------------------

- Der Fehler, dass die Option Sammler unzerstörbar im Gefechtsmodus eingeschaltet sein konnte, wurde behoben.

- Die Reaktionszeit der Einheiten zu Beginn des Spiels wurde verbessert.

- Wenn der Host eines Internetspiels eine veränderte oder eigene Karte benutzt, erhält der Gast jetzt eine Warnung anstatt einer einfachen Vorschau.

- Die gelegentlichen Fehler "Paket zu spät erhalten" oder "Verbindungsabbruch" am Ende eines Internetspiels wurden behoben.

- Die möglichen Fehler "Paket zu spät erhalten" oder "Verbindungsabbruch" bei einem LAN-Spiel nach einem Internetspiel wurden behoben.

- Der Fehler, dass es möglich war, mit verschiedenen Versionen ein LAN-Spiel mit den daraus entstehenden Problemen zu starten, wurde behoben.

- Der Fehler, dass der Spieler im Bau-Modus hängen blieb, wenn er seinen letzten Bauhof verloren hatte, wurde behoben.

- Es gibt einen neuen Regler für die Spielgeschwindigkeit im Internet-Setup-Bildschirm. Hiermit können die Spieler die anfängliche Spielgeschwindigkeit festlegen. Außerdem gibt es jetzt einen Spielgeschwindigkeitsregler in den Spieloptionen, damit er einfacher zugänglich ist. 

- Das Layout des Mehrspieler-Bildschirms wurde verbessert (Zufällige Karte befindet sich jetzt unter der Multiplayer-Karte).
 
- Auf dem Bildschirm Kartenauswahl befindet sich jetzt eine Kartenvorschau.

- Eine neue Latenz-Toleranz-Einstellung im Optionsbildschirm ermöglicht dem Spieler, die Toleranz für Netzwerk-Latenz bei langsamen Verbindungen zu erhöhen. 

- Es gibt eine neue Einstellung auf dem Regler für die Spielgeschwindigkeit (jetzt gibt es die Einstellung 45 fps). 
  Im Multiplayer-Spiel sind jetzt folgende Einstellungen auf dem Schieberegler verfügbar: 
  15 fps (minimal) 20 fps (Standard) 30 fps 45 fps (neu) 60 fps (maximal). 

- Verbesserte Routine zur Bewertung und Veränderung von Verbindungen mit hoher oder geringer Latenz. Im Spiel wird jetzt die Ping-Zeit aus dem Lobby-/Spieloptionsbildschirm benutzt, um die Multiplayer-Zeitwerte für das Spiel festzulegen. Bitte beachten Sie, dass das Spiel jetzt zu Beginn schneller läuft (wenn Sie eine gute Verbindung haben). 

- Neue Routine zur Reduzierung der Bandbreite, wenn die Spielgeschwindigkeit bei 30 fps (eine Einstellung über Standard) oder höher liegt. 

- Allgemein reduzierte Bandbreite bei Multiplayer-Spielen. Zusammen mit dem oben genannten Feature, benötigt das Spiel jetzt viel weniger Netzwerk-Bandbreite im Vergleich zu früher.

- Der Fehler, dass die Spiele wegen bestimmter Abwehrtürme nicht mehr synchron waren, wurde behoben.

- Ein Fehler in der Modem-Verbindungsabbruch-Routine nach einem Spiel wurde behoben.

- Neue "lokale" Einstellungen beim Login-Dialog, so dass Sie Ihre geografische Lage einstellen können. Das wird bei zukünftigen neuen Features auf unserer Website und in der Turniertabelle zum Tragen kommen. 

- Neuer Dialog zur Server-Auswahl. Vielen Spielern war nicht klar, dass sie einen anderen Server als den Westwood Online-Server, zu dem sie zunächst verbunden waren, wählen können. Um dieses Problem zu beheben, haben wir den Dialog "Server-Auswahl" direkt nach dem Login-Dialog eingefügt. Wenn Sie den Server wechseln, erinnert sich das Spiel daran und wählt diesen Server, wenn Sie sich beim nächsten Mal einloggen. 

- Neue Option "Kurzes Spiel" hinzugefügt. Bei diesem Modus hat man verloren, wenn alle Gebäude verloren sind. Das MBF zählt bei Spielende als Gebäude.

- Bis zu 32 Spitznamen können jetzt im Login-Dialogfenster gespeichert werden. 

- Sofortiges Feedback bei einem Befehl im Multiplayer-Modus. 

- Die Sammler können jetzt zu einer anderen Raffinerie umgeleitet werden. 

- Die anderen Spieler werden jetzt benachrichtigt, wenn die Spielgeschwindigkeit oder die Latenzeinstellungen verändert werden. 

- Bei Internet- und Netzwerkspielen wird jetzt ein CRC-Check durchgeführt, da einige Benutzer bestimmter Netzwerkkarten fehlerhafte Pakete erhielten. (Wir konnten das zwar nicht rekonstruieren, aber die Windows-Treiber für RealTek-Karten scheinen daran schuld zu sein.) 

Änderungen in V. 1.13
------------------

- Nod-Artillerie: Feuerrate reduziert

		Weniger effektiv gegen Infanterie
