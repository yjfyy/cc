2.使用说明:

     2.1.XP及Win7系统需要安装.net 4.0以上版本（XP最高支持4.0）,Win10可直接运行。

     2.2.如有新版本会提示更新，如果碰到任何问题，可以强制更新修复。如需更新动画，可选中更新动画，然后强制更新。

     2.3.使用GDI任务、诺德任务、隐秘行动、游戏机独占任务进入游戏，可以分别玩原版GDI、诺德、隐秘行动资料片任务及游戏机独占任务。如果不对应会出现汉化不完全的问题。
